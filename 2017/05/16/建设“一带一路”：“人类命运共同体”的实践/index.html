建设“一带一路”：“人类命运共同体”的实践

发表于 2017-05-16

本文为形势与政策课程作业

人类命运共同体指在追求本国利益时兼顾他国合理关切，在谋求本国发展中促进各国共同发展。人类命运共同体这一全球价值观包含相互依存的国际权力观、共同利益观、可持续发展观和全球治理观。推动建设人类命运共同体，是中国领导人基于对世界大势的准确把握而贡献的“中国方案”。而“一带一路”建设，则是中国构建“人类命运共同体”的伟大实践。

2017年5月14日，习近平主席在“一带一路”国际合作高峰论坛上发表《携手推进“一带一路”建设》的主题演讲，他讲到，“‘一带一路’倡议顺应时代潮流，适应发展规律，符合各国人民利益，具有广阔前景”。“适应发展规律”即是人类命运共同体的可持续发展观，“符合各国人民利益”，即是人类命运共同体中的共同利益观。
历史上，古丝绸之路绵亘万里，延续千年，积淀了以和平合作、开放包容、互学互鉴、互利共赢为核心的丝路精神，是人类文明的宝贵遗产。古丝绸之路沿线地区曾经是“流淌着牛奶与蜂蜜的地方”，如今很多地方却成了冲突动荡和危机挑战的代名词。

当前，世界经济增长需要新动力，发展需要平衡，贫富差距有待弥合，地区热点动荡，恐怖主义蔓延，是摆在全人类面前的严峻挑战。西方国家贸易保护主义抬头，跨境贸易与投资持续低迷。美国，不仅反对多边贸易体系，而且主张“美国主义；欧盟，不仅遭遇了英国“脱欧”，而且其他几个国家也上演“投票箱大戏”。民粹主义、孤立主义和保护主义情绪甚嚣尘上……

在这种国际大环境下，中国不能独善其身，也不能仅仅靠自己的力量推动世界的发展。构建人类命运共同体是当前世界经济发展的必须措施，“一带一路”是这之中非常重要的一环。“一带一路”沿线很多国家资源丰富但基础设施落后，中国可以利用自己的优势，在沿线及周边国家大力投资基础设施建设、加强国际开放交流、提供资金粮食帮助。受惠国家能提高自己的基础设施水平和经济发展水平，而中国能得到广阔的出口贸易市场和人民币国际化渠道，增大在这一重要地区的影响力。

诚然，“一带一路”建设机遇如此之大，但挑战与机遇并存。沿线国家政治的不稳定较高，存在文化的巨大差异和对华恐惧感，这会影响到中国在其的投资。但是，只要我们深入贯彻创新、协调、绿色、开放、共享的发展理念，就能为为“一带一路”注入强大动力，为人类命运共同体建设带来新的机遇。
